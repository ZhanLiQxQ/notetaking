2.1 指针&动态变量

动态变量：（new出来的变量）编程者在程序运行时显示创建和撤销的变量，在需要时才分配存储区域。存在堆中。

指针：存在栈中。

注：
1.与普通变量不同，动态变量决不能直接访问，而总是通过指针变量访问。
指针变量存在栈中，动态变量存在堆中。指针变量！=动态变量
